# Mark Gospel Tools Index

## Overview
This index catalogs all tools developed for integrating your Mark sermon series insights into the MAGA Christianism and Untold America projects. These tools are designed to extract the subversive theological insights from Mark's gospel and apply them systematically across your five publication streams and audio content.

## Analysis Tools

### [Mark Sermon Analysis Prompt](../prompts/mark_sermon_analysis_v1.0.md)
- **Purpose**: Analyze individual Mark sermons for theological insights
- **Output Location**: `/research/sources/analyses/sermons/mark/[passage-reference].md`
- **Primary Application**: Extract sermon insights that counter MAGA Christianism mutations
- **Key Features**:
  - Imperial context analysis
  - Theological mutation confrontation
  - Gospel narrative elements
  - Publication stream applications

### [Mark Thematic Synthesis Prompt](../prompts/mark_thematic_synthesis_v1.0.md)
- **Purpose**: Synthesize insights across multiple Mark sermons into cohesive themes
- **Output Location**: `/research/themes/mark/[theme-name].md`
- **Primary Application**: Develop thematic frameworks based on Mark's theology
- **Key Features**:

  - Imperial context integration
  - Theological mutations confronted
  - Visual and metaphorical framework

## Usage Workflow

1. **Analysis Phase**
   - Use Mark Sermon Analysis Prompt to analyze individual sermons
   - Use Mark Thematic Synthesis Prompt to identify patterns across sermons

2. **Development Phase**
   - Use Mark Publication Conversion Prompt to create stream-specific content
   - Use Mark Visual Concept Prompt to develop supporting visual assets
   - Use Mark Homily Creation Prompt to create audio content

3. **Planning Phase**
   - Use Mark Integration Timeline Prompt to schedule content releases
   - Coordinate across publication streams for thematic coherence
