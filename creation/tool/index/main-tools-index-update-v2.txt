# Main Tools Index

## Overview
This index catalogs all tools developed for the MAGA Christianism and Untold America projects. These tools are designed to streamline research, analysis, synthesis, and publication processes across text, visual, and audio formats.

## Core Analysis Tools

### [Source Analysis Prompt](../prompts/source_analysis_v2.2.md)
- **Purpose**: Generate structured analyses of primary and secondary sources
- **Output Location**: `/research/sources/analyses/[source-type]/[source-name].md`
- **Primary Application**: Extract key insights from research materials
- **Last Updated**: 2025-03-30

### [Thematic Analysis Prompt](../prompts/thematic_analysis_v2.1.md)

## Usage Workflow

1. **Research Phase**
   - Use Source Analysis Prompt to analyze individual sources
   - Use Thematic Analysis Prompt to synthesize across sources
   - Use Mutation Analysis Prompt to track theological mutations

2. **Development Phase**
   - Use Publication Adaptation Prompt to create stream-specific content
   - Use Lexicon Entry Prompt to develop theological vocabulary
   - Use Visual Design Prompt to create supporting visuals
   - Use Audio Content Prompt to develop audio material

3. **Planning Phase**
   - Use Content Calendar Prompt to schedule publications
   - Use Audience Analysis Prompt to tailor for specific readers
   - Use Multi-Channel Strategy Prompt to coordinate across formats

4. **Tool Development Phase**
   - Use Claude Instruction Prompt to create new specialized tools
   - Use Repository Management Script to maintain organization
