# Main Tools Index

## Overview
This index catalogs all tools developed for the MAGA Christianism and Untold America projects. These tools are designed to streamline research, analysis, synthesis, and publication processes across text, visual, and audio formats.

## Core Analysis Tools

### [Source Analysis Prompt](../prompts/source_analysis_v2.2.md)
- **Purpose**: Generate structured analyses of primary and secondary sources
- **Output Location**: `/research/sources/analyses/[source-type]/[source-name].md`
- **Primary Application**: Extract key insights from research materials
- **Last Updated**: 2025-03-30

### [Thematic Analysis Prompt](../prompts/thematic_analysis_v2.1.md)
- **Purpose**: Synthesize insights across multiple source analyses
- **Output Location**: `/research/themes/[theme-category]/[specific-theme].md`
- **Primary Application**: Develop thematic frameworks for chapters and publications
- **Last Updated**: 2025-03-30
